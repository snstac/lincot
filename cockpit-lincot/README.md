# Cockpit Application for LINCOT

This [Cockpit](https://cockpit-project.org/) application manages [LINCOT](https://lincot.rtfd.io): service control, `/etc/default/lincot` configuration, and journal output in a web UI.

## Installing

Install the `.deb` from GitHub Releases, or run `make install` to install under `/usr/local/share/cockpit/`.

Reload Cockpit — **Edge Position (LINCOT)** appears under Tools.
